🆘 มีปัญหา?

  ❌ Port 3000 ถูกใช้แล้ว:
     → เปิด Terminal: npm run dev -- --port 3001

  ❌ AI ไม่ตอบกลับ:
     → ใช้ Mock Mode ได้เลย (ไม่ต้อง API Key)
     → หรือตั้งค่า API Key ที่ /ai-settings

  ❌ Dev server ไม่เริ่ม:
     → รัน: npm install
     → แล้วลองอีกครั้ง: START-DEV.bat

  ❌ Vite cache issues:
     → ลบ: node_modules/.vite
     → รันใหม่: npm run dev
